# 📘 EXERCÍCIOS – Prática do Bootcamp WoMakersCode

Esta pasta reúne os **exercícios sugeridos pela WoMakersCode** ao longo do Bootcamp de BI, com o objetivo de praticar e fixar os conteúdos vistos em aula.  
Os exercícios complementam os **desafios/entregáveis** avaliativos, permitindo treinar fundamentos de análise de dados, SQL, Python, Data Wrangling, ETL e visualização. Alguns exercícios são entregáveis, outros são para prática do que foi visto em aula.

---

## 📆 Exercícios 

### Semana 2 – Fundamentos de Análise de Dados - Entregável

Exercícios teóricos e práticos sobre conceitos iniciais de **análise de dados**, variáveis, métricas e interpretação de informações em contexto de negócio. Análise feita no Google Sheets.

- Enunciado/guia dos exercícios:  
  [Fundamentos de Análise de Dados – Semana 2](https://docs.google.com/document/d/18CYvua5fMblisJPGIKaiL7oUqglNuj8qAAZv9MNmTIM/edit?tab=t.0)

### Semana 3 – Fundamentos de Análise de Dados - Entregável

Continuidade dos **fundamentos de análise de dados**, com foco em leitura crítica de tabelas e gráficos, métricas descritivas e interpretação de cenários. Análise feita no Google Sheets.

- Enunciado/guia dos exercícios:  
  [Fundamentos de Análise de Dados – Semana 3](https://docs.google.com/document/d/1GtuIySYnAf0CcrxGOrZWa60C5IC9bG4Y6DEtjZoMDG8/edit?tab=t.0)

### Semana 4 – Data Analysis com Google Sheets - Entregável

Exercícios de **análise de dados no Google Sheets**, trabalhando fórmulas, filtros, gráficos e criação de tabelas para responder perguntas de negócio. 

- Planilha de referência:  
  [Data Analysis com Google Sheets – Semana 4](https://docs.google.com/spreadsheets/d/1CVwFnsZS5Nvw6QoJqMW193HU0Gngy5qs1x9_DaPYTss/edit?gid=1003749442#gid=1003749442)

### Semana 5 – SQL 1 e 2 

Exercícios focados em **SQL básico e intermediário**:

- Criação de consultas para seleção, filtragem, ordenação e agregação de dados.  
- Primeiros joins e manipulação de tabelas para responder questões de negócio. 

### Semana 6 – Python, pandas e numpy - Entregável

Exercícios para praticar **Python para dados**, com:

- Leitura de dados e criação de DataFrames com `pandas`.  
- Operações com `numpy` para tratamento e transformação de dados.  
- Análise exploratória simples no notebook. 

### Semana 7 – Data Wrangling e Pipeline (Python para Dados)

Prática de **data wrangling** (limpeza, padronização, tratamento de valores ausentes) e início da construção de pequenos **pipelines de dados** em Python para organizar etapas de análise. 

### Semana 15 – Pipeline de ETL Integrado 

Exercícios voltados à construção de um **pipeline de ETL (Extract, Transform, Load)**:

- Extração de dados de fontes simuladas.  
- Transformações estruturadas (limpeza, criação de colunas, padronização).  
- Preparação dos dados para consumo em dashboards e análises posteriores. 

### Semana 16 – Criando Gráficos com Bibliotecas Python

Exercícios de **visualização de dados em Python**, explorando bibliotecas como `matplotlib`, `seaborn` e `ploty`:

- Criação de gráficos de barras, linhas, dispersão, entre outros.  
- Customização de títulos, eixos e cores para comunicação clara de insights. 

---

## 🧠 Objetivo dos Exercícios

Os exercícios desta pasta foram pensados para:

- Reforçar os conceitos apresentados nas aulas semanais.  
- Desenvolver a **prática contínua** em ferramentas de análise de dados (Sheets, SQL, Python).  
- Apoiar a construção de uma base sólida para os **desafios** e **projetos de portfólio** desenvolvidos ao longo do Bootcamp.
